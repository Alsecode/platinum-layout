### <h1>Platinum ProConstruction Layout</h1>
Adaptive and optimized landing for Platinum ProConstruction using HTML, CSS (SASS), JS and BEM methodology for naming.

### Install the dependencies:
```
$ npm install
```
### Start a website:
```
$ npm run dev
```
### Make a build
```
$ npm run build
```
